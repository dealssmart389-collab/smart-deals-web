## 🚀 نظرة عامة (Overview)
هذا المستودع هو القلب النابض لمنظومة **Benalal AI** الموحدة، والتي تجمع بين الذكاء الاصطناعي المرئي، إدارة القوانين المرورية (NARSA)، ومنصة التسويق الذكي (Smart Deals).

## 🧠 المكونات الأساسية (Core Modules)
* **TIMO OMNI-CORE**: المحرك المنطقي الذي يحلل استفسارات المستخدمين حول مدونة السير المغربية.
* **Driving AI 2026**: نظام محاكاة امتحانات السياقة بناءً على بيانات رسمية.
* **Smart Deals Engine**: منصة الربط الذكي مع AliExpress و Jumia لجني أرباح الـ Affiliate.
* **Gatekeeper API**: جسر التواصل المشفر بين واجهة الويب ونظام **Termux** المحلي.

## 🛠️ التكنولوجيا المستخدمة (Stack)
* **Frontend**: HTML5/CSS3 (Matrix Theme), JavaScript (Neural Logic).
* **Backend**: Python 3, Flask (Gatekeeper Service).
* **Automation**: GitHub Actions, Telegram Bot API (@MyDrivingAI_Bot).
* **Environment**: Android Termux (Mobile-First Development).
